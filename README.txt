# Sistema de Cardápio Digital Interativo

## Descrição

Este projeto implementa um sistema de cardápio digital interativo para restaurantes fast food. Utilizando HTML, CSS, JavaScript e Bootstrap, o sistema permite que os clientes visualizem o cardápio, selecionem itens, ajustem quantidades e finalizem seus pedidos online. É uma solução ideal para modernizar o processo de pedidos e melhorar a experiência do cliente.

## Objetivo

O objetivo principal é fornecer uma interface amigável para clientes realizarem pedidos de forma eficiente. O sistema é dividido em categorias como Hambúrgueres, Salgados, Bebidas e Sobremesas, com opções para adicionar itens ao carrinho, ajustar quantidades e escolher métodos de pagamento.

## Funcionalidades

- **Exibição do Cardápio:** Itens organizados em categorias com informações sobre nome, descrição e preço.
- **Seleção de Itens:** Checkboxes para selecionar itens desejados e campos de quantidade para ajuste.
- **Gerenciamento do Pedido:** Adição e remoção de itens no carrinho, com atualização dinâmica do total.
- **Finalização do Pedido:** Opções de pagamento (cartão ou Pix), coleta de informações de contato para entrega e exibição de um resumo do pedido em um modal para confirmação.

## Tecnologias Utilizadas

- **HTML:** Estruturação do conteúdo.
- **CSS:** Estilização e layout do cardápio.
- **JavaScript:** Lógica para gerenciamento de pedidos e atualização dinâmica do carrinho.
- **Bootstrap:** Framework para estilização e responsividade.
